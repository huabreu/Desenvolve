Faturamento Líquido = Faturamento - Custo Frete - Imposto

Custo Total = Custo Fixo - Custo Variavel

Margem = Faturamento Líquido - Custo Total

Faturamento Unitário = SUM(Faturamento Líquido) / SUM(Quantidade Vendida)

Custo Unitário = SUM(Custo Total) / SUM(Quantidade Vendida)

Margem Unitária = SUM(Margem) / SUM(Quantidades Vendida)

Variação Meta Faturamento = ((SUM(Faturamento Líquido) / SUM(Meta Faturamento)) - 1) * 100

Variação Meta Custo = ((SUM(Custo Total) / SUM(Meta Custo)) - 1) * 100

Variação Meta Margem = ((SUM(Margem) / SUM(Meta Margem)) - 1) * 100
